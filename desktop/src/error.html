Viewer Initialization Failed

The LIV viewer process could not be started. This may be due to a missing component or system configuration issue.

Error: Viewer executable not found or failed to start

Troubleshooting Steps:

  • Ensure the LIV viewer executable is installed correctly
  • Check that all required dependencies are available
  • Try running the application as administrator
  • Verify that no firewall is blocking the application
  • Check the application logs for more details

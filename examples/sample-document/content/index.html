What is LIV?

LIV (Live Interactive Visual) is a document format that combines the portability of PDF with modern web technologies. It enables rich interactive content while maintaining security and cross-platform compatibility.

Key Features

  • 📦 Single-file container with all assets
  • 🔒 Secure sandboxed execution
  • ✏️ Editable content with WYSIWYG tools
  • 🎬 Live animations and interactive elements
  • 📱 Cross-platform compatibility
  • 🔄 Format conversion capabilities
